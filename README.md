# Quiz Application

A web-based quiz application built using HTML, CSS, JavaScript, and Bootstrap.
This is a timer-based quiz project that displays MCQs and shows the final score and rank.

## Features
- Multiple Choice Questions (MCQs)
- Timer-based quiz
- Previous and Next navigation
- Score and rank calculation
- Sound effects for answers
- Responsive design using Bootstrap

## Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap 5

## How to Run the Project
1. Clone or download the repository
2. Open the project folder in Visual Studio Code
3. Right-click on index.html
4. Select "Open with Live Server"
